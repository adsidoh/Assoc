===========
Dossiers
===========
=> Associations: contient toutes les vues html appartenant au controller Associations.
=> Comptas: contient toutes les vues html appartenant au controller Comptas.
=> Elements: non utilisé car Theme Cakestrap utilisé, donc voir Themed/Cakestrap/Elements.
=> Emails: non utilisé.
=> Errors: permet de spécifier les erreurs (base de données, php, html) lorsque mode debug à 2 (voir ../Config/core.php line 34).
=> Fournitures: contient toutes les vues html appartenant au controller Fournitures.
=> Helper: non utilisé.
=> Layouts: non utilisé car Theme Cakestrap utilisé, donc voir Themed/Cakestrap/Layouts.
=> Pages: non utilisé, contient page d'accueil par défaut.
=> Recherches: contient toutes les vues html appartenant au controller Recherches.
=> Responsables: contient toutes les vues html appartenant au controller Responsables.
=> Scaffolds: non utilisé.
=> Students: contient toutes les vues html appartenant au controller Students.
=> Themed: contient tous les éléments spécifique à un thème utilisé, dans notre cas voir dossier Cakestrap.
=> Users: contient toutes les vues html appartenant au controller Users.
